## Instrument manual overview

Tenney documents the lattice, tuner, and library workflows used in live tuning work. Use these notes as an instrument manual for rehearsals, installations, and studio sessions.

### Table of contents

- Lattice orientation and grid traversal
- Limit profiles and root selection
- Tuner modes and lock workflows
- Saving systems and exporting Scala files

### Quickstart

1. Choose a root pitch and limit.
2. Traverse ratios and select points on the lattice.
3. Save the set and export when ready.
